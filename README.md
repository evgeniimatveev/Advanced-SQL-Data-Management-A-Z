## üìÇ Project Structure:

### **Section 1: SQL Basics**

- `TCL+DCL(S1).sql` ‚Äî *Transaction Control Language (TCL) and Data Control Language (DCL) examples.*

### **Section 2: Intermediate SQL**  
Dive into JOINs, GROUP BY, HAVING, subqueries, and string functions.
- `Aggregate_Functions(S2).sql` ‚Äî *Working with aggregate functions such as `SUM()`, `AVG()`, etc.*
- `Final_Project_Script_part_2(S2).sql` ‚Äî *Intermediate steps and data processing in the final project.*

- `SELECT_Disign(S2).sql` ‚Äî *Advanced SELECT query examples.*
- `String_Functions(S2).sql` ‚Äî *Functions for string manipulation.*
- `SubQuery(S2).sql` ‚Äî *Example of subqueries.*

### **Section 3: Advanced SQL**  
Explore complex SQL topics including window functions, PIVOT, and query optimization.
- `Window_Functions(S3).sql` ‚Äî Using window functions for advanced data analysis.
- `PIVOT_in_MYSQL(S3).sql` ‚Äî Pivoting data in SQL.
- `UNPIVOT_in_MYSQL(S3).sql` ‚Äî Unpivoting data in SQL.
- `Indexes(S3).sql` ‚Äî Creating and using indexes for optimization.
- `Query_Optimization_Techniques(S3).sql` ‚Äî Tips for optimizing SQL queries.
